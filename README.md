# 基于springboot的职称评审系统3



## 1、项目介绍

基于springboot的职称评审系统3拥有两种角色

管理员：参评人（教师）管理、专家管理、专业管理、职称管理、评议管理、评级结果管理、通知管理等

专家：查看通知、评审打分、已评管理等

## 2、项目技术

后端框架：spring

前端技术：jsp

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：Tomcat 7-10版本均可
- maven版本：无限制
- 硬件环境：Windows
